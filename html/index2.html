O que é?

CPU (Central Processing Unit / Unidade Central de Processamento) é um tipo de chip eletrônico especializado.

Sua função é acelerar, endereçar, resolver ou preparar dados, dependendo da aplicação. Basicamente, um processador é uma potência de computador.

O componente recebe um determinado volume de dados, orientado segundo um modelo binário de 0 e 1, e tem a função de responder a esse volume, processando as informações de acordo com as instruções armazenadas em sua memória interna.

História

Na década de 1940, época do surgimento da computação eletrônica.

Durante este período, os computadores usavam manualmente válvulas termiônicas ou tubos de vácuo para realizar lógica, processamento de dados e outros tipos de operações.

A década de 1950 viu a invenção do transistor, um componente eletrônico semicondutor muito menor e com maior eficiência energética. Com esse projeto, tornou-se possível desenvolver computadores menores e muito mais rápidos, como o IBM 7090.

Na década de 1960, surgiram no mercado circuitos integrados compostos por diversos transistores e outros componentes eletrônicos integrados em uma única unidade. Essa inovação possibilitou a miniaturização dos computadores e aumentou a capacidade de processamento.

O Intel 4004 de 1971 foi um exemplo de microprocessador de circuito integrado.

Em 1978, foi lançado o Intel 8086, um dos primeiros microprocessadores de 16 bits amplamente uti lizados em computadores pessoais. A empresa lançou então outros processadores até chegar ao Pentium, que se tornou muito popular nas décadas de 1980 e 1990.

Como funciona?

Os processadores de computador trabalham com outros componentes de hardware e software para processar dados e gerenciar o fluxo de informações entre dispositivos eletrônicos.

Geralmente funcionam em ciclos, sendo que cada ciclo de instrução representa três fases principais:

1. Ciclo do processador principal:

As etapas básicas de um loop de instrução são mostradas abaixo.

2. Encontre a direção:

A CPU recupera instruções da memória. As instruções são códigos binários que representam tarefas ou operações específicas do processador.

A unidade de controle interpreta as instruções e determina a operação a ser executada. Ele também identifica os componentes específicos da CPU necessários para a tarefa.

3. Instruções de processamento:

O processador executa a operação especificada nos dados recebidos. Executa cálculos matemáticos, comparações lógicas, manipulação de dados ou transferência de dados entre registros ou locais de memória.

4. Salve os resultados:

Após executar as instruções, o processador pode precisar armazenar os resultados na memória ou atualizar registros especiais com os novos dados. O contador de programa (PC) é atualizado para indicar o endereço da próxima instrução a ser buscada.

A CPU repete o ciclo e busca, decodifica e executa instruções sequencialmente.

Curiosidades

Aqui vão algumas curiosidades rápidas sobre o processador:

1. APU:

APU vem de Accelerated Processing Unit (Unidade de Processamento Paralelo) e é o nome dado a processadores que possuem tanto uma CPU como uma GPU (Unidade de Video)

2. Clock:

Ter mais ou menos Hertz significa o quanto o processador troca dados com o sistema. O processador que oferece 2.0 GHz (Giga Heartz) pode realizar 2 bilhões de ciclos por segundo.

O circuito clock, que mede os ciclos e orienta o ritmo do fluxo de troca de informações no processador, é um dos principais critérios para estabelecer a velocidade do processador.

Vale ressaltar, no entanto, que outros pontos entram nesta conta, como interface de memória, quantidade de cache, arquitetura, entre outros.

3. MMU:

a Memory Management Unit, ou Unidade de gerenciamento de memória, é responsável por coordenar as operações de memória.

O processador só pode ser rápido se a RAM continuar. A MMU é o recurso que transforma instruções lógicas (virtuais) em endereços físicos nos bancos de memória.

4. Cache:

Dependendo da quantidade de trabalho que a CPU realiza, as informações solicitadas são constantemente distribuídas nesta área.

Isso é feito para economizar tempo. Armazenados no processador, esses dados podem ser acessados rapidamente e não há necessidade de escanear o disco ou a RAM para encontrar as informações.
